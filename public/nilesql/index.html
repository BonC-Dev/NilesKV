NilesQL

A query language for NilesKV. Filter documents by key pattern and field conditions. Runs on top of the field index built at commit time.

nilesql query
cmd+enter

Query syntax
GET *
Return every document in the store.
GET user:*
Return all documents whose key starts with user:.
GET user:alice
Return the single document at the exact key.
GET user:* WHERE role = "admin"
Equality filter on a string field.
GET order:* WHERE amount > 100
Numeric comparison: > < >= <= != =
GET user:* WHERE role = "admin" AND active = true
Chain conditions with AND.
